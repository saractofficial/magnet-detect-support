# magnet-detect-support
magnet-detect-support
磁力検知 サポートページ

アプリ「磁力検知」をご利用いただきありがとうございます。

■ ご利用方法
1. アプリを起動し、「▶ 計測開始」をタップしてください
2. 磁石を近づけると数値が変化します
3. 一定値を超えると振動でお知らせします

■ よくあるご質問

Q. 数値が常に0のまま動きません
A. Apple Watch Series 5以降が必要です。また「▶ 計測開始」を必ずタップしてください。

Q. 振動が鳴りません
A. Watch のおやすみモード／消音モードを確認してください。
   閾値は「GPS自動ON設定」内のスライダーで調整できます。

Q. GPS自動ONが動きません
A. 設定画面で位置情報の利用を「使用中のみ許可」にしてください。
